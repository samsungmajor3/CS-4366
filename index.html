Description

The project aims to assess renewable energy availability at the GLEAMM facility and implement workload control using an API on the test cluster's head node. The focus is on retrieving real-time power data, predicting available power, and dynamically scheduling jobs on the test cluster to align with renewable sources and backup battery levels. The API can also remotely power on/off nodes, optimizing energy usage and extending battery run time as needed.

Benefits

This project can provide the benefits of optimizing operational costs and minimizing carbon emissions by implementing an innovative solution. Also, the system is designed to be scalable and capable of meeting the demands of large-scale data facilities. Through intelligent workload control, the project ensures the efficient utilization of renewable energy sources, contributing to environmental sustainability. Moreover, the implementation allows for the testing of cluster operation limits, enabling a comprehensive evaluation of system capabilities.
