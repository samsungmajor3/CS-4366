Description

The project aims to assess renewable energy availability at the GLEAMM facility and implement workload control using an API on the test cluster's head node. The focus is on retrieving real-time power data, predicting available power, and dynamically scheduling jobs on the test cluster to align with renewable sources and backup battery levels. The API can also remotely power on/off nodes, optimizing energy usage and extending battery run time as needed.

Go to Github Source

Objective

The purpose of this project is to use renewable energy efficiently by making an API to control the workload on a test cluster at the GLEAMM facility. Depending on the availability of renewable energy resources, the API will control jobs on a test cluster. By controlling a workload and being able to make use of renewable energy, we will be able to meet the demand for more processing capability nowadays while lowering its emissions contributions. The intended audience of this project is the faculty members and PhD students working in the GLEAMM facility.
